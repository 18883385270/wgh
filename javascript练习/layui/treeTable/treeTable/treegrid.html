TreeGrid(树形表格)

依赖模块：layui.grid
语法：layui.treegrid.createNew(option);
说明：treeGrid继承于grid组件，拥有grid的所有属性、方法、事件。标注"替换"的方法，在treeGrid中请勿使用。

属性
属性名称 值类型 默认值 说明
id string 'id' 节点id字段
parentid string 'parentid' 父节点id字段
root string/number 0 根节点值
order string =id 排序字段，默认为id字段
asc string 'asc' 排序方式，顺序排序“asc”，倒序排序“desc”
方法
方法名称 参数类型 返回值类型 说明
collapse(id) int/string/undefined 无 折叠节点，若参数为undefined,折叠一级节点
collapseAll(id) int/string/undefined 无 折叠节点及其子节点，若参数为undefined,折叠全部节点
expand(id) int/string/undefined 展开节点，若参数为undefined,展开一级节点
expandAll(id) int/string/undefined 无 展开节点及其子节点，若参数为undefined,展开全部节点
getChild(id) int/string array 根据节点ID，获取节点的下级节点
getChildAll(id) int/string array 根据节点ID，获取节点的所有子节点
insertNode(rows,id) rows:object/array
id:int/string/undefined 		无 在节点之前添加一个或多个同级节点，rows数据可为子父级关系的数据，替换insert方法
insertChild(rows,id) rows:object/array
id:int/string/undefined 		无 添加一个或多个子节点。rows数据可为子父级关系的数据
updateNode(row,id) row:object
id:int/string		 无 修改节点，替换update方法
deleteNode(id) int/string 无 删除节点及其所有子节点
editNode(id) int/string 无 编辑节点，若id为undefined,编辑整个表格，替换edit方法
endEditNode(id) int/string 无 结束节点编辑，若id为undefined,整个表格结束编辑，替换endEdit方法
事件
事件名称 参数说明 说明
loadRow(data) data={
     id:id, //节点id
     children:function(rows) //添加子节点的回调函数
} 		懒加载，设置此事件，表格数据行中字段'_children'值为'true'的行将开启懒加载，
回调函数的rows参数，可以为子父关系的数据
